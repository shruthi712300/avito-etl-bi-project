# Avito ETL & BI Project

## Overview
End-to-end data engineering project using Azure Data Factory, Azure Blob Storage, and Power BI.

## Architecture
- Raw → Processed → Curated data layers
- Parquet used for curated layer
- Power BI dashboard built on curated data

## Tech Stack
- Azure Data Factory
- Azure Blob Storage
- Azure VM (Linux)
- Power BI
- GitHub
